# todo-project

這是個人小作品，一個簡單實用的 Todolist 應用介面，專門用來記錄日常事項。支援編輯、修改、刪除與勾選完成等功能，協助有效管理瑣碎日常，提升生活效率。

---

## 功能介紹

- 新增待辦事項
- 編輯與修改事項內容
- 勾選完成 / 未完成狀態
- 刪除不需要的事項
- 簡潔清爽的 UI 設計

---

## 使用技術與套件

| 套件 | 用途 |
|------|------|
| [Vue 3] | 建構前端應用的核心框架 |
| [Vercel] | 自動部署平台 |
| [Tailwind CSS] | 實用為主的 CSS 框架 |
| [daisyUI] | 基於 Tailwind 的 UI 元件庫 |

---

## 快速開始

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

---

## 部署方式

本專案使用 [Vercel] 自動部署，只需連接 GitHub 倉庫，Vercel 會自動建置與部署。
